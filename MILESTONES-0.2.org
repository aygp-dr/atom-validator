* Validation & Deployment (User + Claude)

Each milestone requires:

1. =make check= - all tests pass
2. =make security= - lint + nvd
3. =bin/validate-deploy= - JAR audit
4. =make verify-publish= - E2E Clojars check
5. =git-cliff= - changelog generation
6. =gh release create= - tagged release with notes
7. cljdoc trigger - documentation rebuild

* Success Criteria

- [ ] CLI usable from shell scripts (exit codes, JSON output)
- [ ] Babashka one-liner: =bb -e "(require '[atom-validator.core :as v]) ..."=
- [ ] MCP tool callable from Claude Code
- [ ] Native binary <20MB, starts <100ms
- [ ] GitHub Action published to marketplace
- [ ] E2E publish verification on every release
